Snake Game

Jogo clássico da cobrinha desenvolvido em JavaScript.

Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

Objetivo

Praticar lógica de programação e controle de eventos do teclado.

O que aprendi

- Manipulação de canvas
- Controle de movimentação com teclado
- Lógica de colisão
- Estruturação de loops e condicionais

Como executar

1. Abra o arquivo `index.html`
2. Use as setas do teclado para jogar
